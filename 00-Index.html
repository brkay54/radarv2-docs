radarv2 — AERIS-10 Pulse-LFM Phased-Array Radar (aros example)¶

What this is

examples/radarv2 is an aros reference example that models the AERIS-10

open-source 10.5 GHz pulse-LFM phased-array radar as a graph of aros nodes.

Where the existing radar example is an abstract tracking

pipeline, radarv2 reproduces AERIS-10's actual **receive signal-processing

chain** — digital down-conversion, matched-filter pulse compression, Doppler

FFT, and CFAR detection — plus its scan/beam control and GPS/IMU telemetry,

with every interface defined as an aros message.

Upstream source: github.com/mitgor/PLFM_RADAR (project name AERIS-10),

cloned and read in full. This vault declares the algorithms, flows, and

messages; the implementation lives in examples/radarv2/.

One-paragraph summary¶

AERIS-10 transmits linear-FM chirps through a 16-element electronically-steered

X-band array, sweeps 31 elevation beam positions per azimuth step, and mechanically

rotates through 50 azimuth positions for a full revolution (~9 s, ~0.11 Hz). Each

beam dwell coherently integrates M = 32 chirps. The receive echo is digitized

at 400 MSPS, down-converted to complex baseband, pulse-compressed against the

reference chirp into a 64-bin range profile, Doppler-processed with a 32-point FFT

into a 64×32 range-Doppler map, thresholded for detections, tracked, and streamed to a

host. radarv2 reproduces this end to end — through detection, plot conditioning, and

tracking — as eleven aros nodes across seven DKMs in the single-radar spine (a multi-radar

fleet adds per-radar fusion + control nodes), exercising

all four aros transports (DirectHandoff, LocalChannel, SHM, UDP/TCP) on a faithful

but tractable software model — and overlays real **DTED terrain + road/coastline

maps** (MapProvider) for terrain-aware clutter, masking,

and visualization.
